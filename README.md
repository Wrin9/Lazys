usage: app.py [-h] [--mode MODE] [--search SEARCH] [--start START] [--end END]

options:
  -h, --help       show this help message and exit
  --mode MODE      Search mode
  --search SEARCH  Search query
  --start START    Start, default 0
  --end END        End, default 10
